## Usage

![](./assets/alfred-projj.gif)

1. Install [projj](https://github.com/popomore/projj)

```bash
$ npm i projj -g
$ projj init
```

2. Install [Alfred](https://www.alfredapp.com/workflows/)

3. Install Workflow:

```bash
$ npm i alfred-plugin-projj -g
```
